# Track_bot 
> Проект Track_bot это программа детектор настроения, которое показывает настроение пользователя опираясь на его записи.

Репозиторий содержит код Telegram-бота, разработанного в рамках колледж-проекта.

---

## Инструкция по запуску

### 1. Подготовка окружения и установка зависимостей

Выберите инструкцию для вашей операционной системы:

####  Windows
1. Создайте виртуальное окружение:

   python -m venv venv

2. Активируйте виртуальное окружение:

   venv\Scripts\activate

3. **Установите необходимые библиотеки:**

   pip install -r requirements.txt


####  Linux /  macOS
1. **Создайте виртуальное окружение:**

   python3 -m venv venv

2. **Активируйте виртуальное окружение:**

   source venv/bin/box/activate

   (Если используете оболочку fish, csh или tcsh, используйте соответствующий скрипт активации, например source venv/bin/activate.fish)
3. **Установите необходимые библиотеки:**

   pip install -r requirements.txt


### 2. Настройка конфигурации (API Токен)

1. Откройте Telegram и найдите официального бота @BotFather(https://t.me).
2. Создайте нового бота с помощью команды /newbot, задайте имя и юзернейм.
3. Скопируйте полученный API Token.
4. В корневой директории проекта создайте файл с именем .env.
5. Добавьте в файл .env следующую строку, вставив свой токен:
   
   TOKEN="тут_ваш_токен_от_BotFather"


### 3. Запуск бота
После успешной активации окружения и настройки .env файла, запустите главный скрипт:


python bot.py

(Для Linux/macOS может потребоваться команда python3 bot.py)
